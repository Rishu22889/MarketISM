**MarketISM** is a campus-exclusive marketplace app built for **IIT (ISM) Dhanbad** students.  
It allows students to **buy and sell used items** such as books, gadgets, and essentials **safely within the campus**.  
Built using **Flutter** and **Supabase** for authentication and backend.

- **Secure Authentication**

  - Upload product photos, titles, prices, and descriptions.
